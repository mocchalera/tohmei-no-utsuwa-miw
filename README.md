# 透明の器 — Music Interactive Web (v2)

楽曲「透明の器」のための MIW（MVではなく、聴きながら触れる一枚の窓ガラス）。

## コンセプト

**あなたは雨の朝、誰もいない街の、ガラスの内側にいる。**

画面そのものが「透明の器」＝窓ガラス。

- ガラスは朝もやで曇っている。なぞると拭ける。放っておくと、また静かに曇る
- あまつぶは結露し、近くの滴と**くっついて**大きくなり、通り道の滴を**とりこんで**、重くなると**落ちてゆき**、画面下の**水たまりの中に消える**（波紋）
- しずくに触れると「数えられる」。数字がふっと浮かび、隅のカウンターが増える — **あまつぶ数えて暇つぶす**
- 何もない場所に触れると、吐く息でガラスが曇り、新しい結露が生まれる — **自分の吐く息と雨の音だけ**
- しばらく触れずにいると、画面の下のほうが呼吸のリズムで曇る
- 曲が終わると曇りが晴れ、数えたあまつぶの数が残る

## 音楽連動

- 曲のエネルギー → 雨（結露）の量
- 低域のビート → 滴の滑落・波紋の広がり
- 歌詞タイムライン → 演出イベント（カラスの群れ / 滴の引き寄せ / 一斉落下 / 水たまりの波紋）

## 起動

```sh
cd 03_works/tohmei-no-utsuwa-miw-v2
python3 -m http.server 8000
# → http://localhost:8000
```

`file://` 直開きでも動くが、ブラウザによっては音解析が無音になるため、その場合は擬似エネルギーに自動フォールバックする。サーバー経由を推奨。

## 操作

画面下のトランスポートバー（普段はもやに沈み、マウスを下端に近づけると浮かぶ）:

- **再生 / 一時停止** … ボタン、または **スペースキー**
- **シーク** … バーをクリック／ドラッグ。後方へ飛ぶと歌詞も追従して巻き戻る
- **音量 / ミュート** … 右側のスライダーとアイコン

そのほか:

- ガラスをなぞる → 曇りを拭く
- しずくに触れる → 数える
- 何もない場所に触れる → 吐く息で曇り、新しい結露が生まれる

## 歌詞タイミング

歌詞は `main.js` 冒頭の `TIMELINE` 配列（秒）。値は **faster-whisper の単語タイムスタンプから実測**し、各行を発声の約0.4秒手前で立ち上げている（フェードイン1.3秒でボーカルに重なる）。

再計測したい場合:

```sh
python3 - << 'EOF'
from faster_whisper import WhisperModel
m = WhisperModel("small", device="cpu", compute_type="int8")
segs, _ = m.transcribe("assets/tohmei_no_utsuwa.mp3", language="ja", word_timestamps=True, vad_filter=False)
for s in segs:
    for w in (s.words or []):
        print(f"{w.start:7.2f}  {w.word}")
EOF
```

微調整用に、再生中 **L キー**で現在時刻をコンソール出力する。

## 構成

```
index.html   画面・オーバーレイ
style.css    縦書き歌詞・スタート/終景
main.js      雨滴エンジン・曇り層・音解析・タイムライン
assets/tohmei_no_utsuwa.mp3
```

依存ライブラリなし（Canvas 2D + Web Audio API のみ）。
